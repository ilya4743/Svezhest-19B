Обновление прибора

Прибор “Свежесть 19B” разработан для анализа качества воздуха под условия каждодневной работы в учебных аудиториях МФ МГТУ им. Баумана. Его очень удобно использовать в качестве индикатора, дающего качественную оценку газовой смеси, осуществляя контроль воздуха рабочей зоны.

Измеряемые параметры:

  • Температура;
  • Влажность;
  • Давление;
  • Эквивалент углекислого газа (eCO2, pm);
  • Общее количество летучих органических компонентов (TVOC, ppm);
  • Углекислый газ (CO2, ppm).
В состав прибора входят следующие датчики:
  • MH-Z19B;
  • CCS811;
  • BME280.
Для удобного управления прибором было разработано настоящее веб-приложение. Веб-приложение имеет следующие функции:
  • Переключение режимом LCD экрана;
  • Управление подсветкой LCD экрана;
  • Конфигурирование сетевых настроек;
  • Удаленная перепрошивка прибора;
  • Доступ к файловой системе прибора.
Переключение режимов LCD экрана

LCD экран имеет 6 режимов отображения:
  • ПОКАЗАНИЯ (BME280, CCS811, MH-Z19B);
  • IP ;
  • BME280;
  • CCS811;
  • MH-Z19B;
  • ВСЁ (IP, BME280, CCS811, MH-Z19B).

Для перехода на тот или иной режим необходимо лишь нажать соответствующую кнопку (см. рис. ) Каждый 5 секунд экран будет выводить следующую страницу или же обновлять текущую, в зависимости от выбранного режима.

Управление подсветкой LCD экрана

В целях энергосбережения есть возможность отключения подсветки LCD экрана (см. рис. )

Конфигурирование сетевых настроек

При включении прибор попытается подключиться к точке доступа, параметры для подключения к которой хранятся в конфигурационном файле(configs.json). Если подключение по каким-то причинам не удалось, то будет поднята точка доступа прибора.

Для конфигурирования сетевых настроек необходимо подключиться к той же сети, что и прибор или же подключиться к его точке доступа и перейти на соответствующую страницу настроек(см. рис.) Пользователь может указать сеть, к которой необходимо подключиться, может изменить название и пароль точки доступа прибора прибора.

Все изменения будут внесены в конфигурационный файл(configs.json).

Удаленная перепрошивка прибора

Для перепрошивки прибора необходимо перейти на страницу “Обновление прибора”. Учитывая опасность для работоспособности при очередном обновлении доступ к странице ограничен и требует ввода пароля, который есть только у администратора. Чтобы выполнить обновление, нужен скомпилированный двоичный файл новой прошивки. В Arduino IDE такой файл можно получить, пройдя по следующему пути “Sketch/Export compiled Binary” (см. рис.). Когда двоичный файл получен необходимо лишь выбрать его при помощи интерфейса веб-приложения прибора (см. рис) и нажать кнопку “Обновить” (см. рис).

Доступ к файловой системе прибора

Для доступа к файловой системе прибора необходимо перейти на страницу “Обновление прибора” и нажать кнопку “Редактор веб-страниц”. Учитывая опасность для работоспособности при работе с файловой системой доступ к странице ограничен и требует ввода пароля, который есть только у администратора. Интерфейс для работы с файловой системой изображен на рис. . Файлы можно редактировать, добавлять, удалять, переименовывать. (см. рис.)
